Stokes Distributing (In Production)

Project brief:

Stokes Distributing is a beer distributer in south Mississippi. This website is full of features such as displaying all of their products, job applications and and an events page among other features creating a great user experience for all their needs.

Project Details:

  • Technologies:Django, Javascript, S3, Bootstrap, SQL, Heroku
